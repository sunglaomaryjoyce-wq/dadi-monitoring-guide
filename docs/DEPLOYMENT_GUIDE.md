# Deployment Guide

Use this package as a clean monitoring-only website project.

## GitHub Upload

Upload the extracted contents of this folder, not the ZIP file itself. The repository root should show:

```text
app/
components/
lib/
public/
docs/
package.json
next.config.mjs
tsconfig.json
README.md
```

## Vercel Settings

- Framework Preset: Next.js
- Root Directory: blank or ./
- Build Command: default
- Output Directory: default
- Install Command: default

## Logo

The logo file is located at:

```text
public/dadi-logo.png
```
